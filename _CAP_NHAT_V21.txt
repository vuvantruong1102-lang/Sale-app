CẬP NHẬT V21 — thêm cột Trạng thái đơn hàng + chống import nhầm file
===================================================================

>>> BƯỚC 1: chạy supabase_migration_v7.sql (thêm cột order_status).

>>> BƯỚC 2: up đè các file, Vercel build.

ĐÃ LÀM:
1. Thêm cột "Trạng thái" vào bảng Đơn hàng (đọc từ cột D của file):
   - Shopee: cột D "Trạng thái đơn hàng"
   - TikTokShop: cột D "Order Status"
   Cột này cũng có trong file Xuất Excel.

2. IMPORT LẠI = CẬP NHẬT trạng thái: khi import lại file có cùng mã đơn,
   trạng thái mới sẽ ghi đè trạng thái cũ (vd "Đang giao" -> "Đã giao").
   Các thông tin khác cũng cập nhật theo file mới. KHÔNG tạo đơn trùng.

3. CHỐNG IMPORT NHẦM FILE: nếu bấm nút Shopee nhưng chọn file TikTokShop
   (hoặc ngược lại), app NHẬN DIỆN và BÁO LỖI, từ chối import — tránh tạo
   dữ liệu rác. Nhận diện dựa trên đặc điểm mã đơn (TikTok: toàn số 15-19
   chữ số; Shopee: chữ-số ~14 ký tự).

FILE CẬP NHẬT (giữ cấu trúc thư mục):
  supabase_migration_v7.sql        (chạy Supabase trước)
  src/lib/parsers.js               (đọc cột D + nhận diện file)
  src/components/ImportPanel.jsx   (chống nhầm + gửi order_status)
  src/pages/Dashboard.jsx          (cột Trạng thái trong bảng + export)
  src/admin.css                    (style tag trạng thái)
  api/admin.js                     (lưu/cập nhật order_status)

GHI CHÚ: bảng "Bảo hành" vốn đã có cột Trạng thái (active/expired) của bảo
hành. Cột mới này là TRẠNG THÁI ĐƠN HÀNG TỪ SÀN, nằm ở bảng Đơn hàng.
Nếu bạn muốn hiện thêm trạng thái sàn ở bảng Bảo hành nữa, báo mình thêm.
